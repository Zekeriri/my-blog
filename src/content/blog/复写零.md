[1089. 复写零 - 力扣（LeetCode）](https://leetcode.cn/problems/duplicate-zeros/description/)
# 题意
将数组中的 0 复写一遍, 将其余元素往右移
# 思路
## 1. 暴力
遍历数组, 遇到 0 时, 将后面的元素后移一位, 在后一位添加一个 0  
时间复杂度为 O(n^2)
## 2. 双指针
### 思路
思考一下怎么从暴力上优化, 暴力算法将后面的元素往后移需要多消耗一层 O(n)  
如果从后往前遍历, 就只用往后移遍历到的元素, 不用移动没有遍历到的  
所以先遍历一遍, 获得 0 的个数 m, 然后给数组新添加 m 位  
设置一个 cur = n-1 指针用来遍历数组,  
再设置一个 dest = n+m, 用来表示已复写数组的第一位 (倒着写的)  
但遍历到非 0 元素时, 复写, cur--, dest--  
遍历到 0 时, 复写两个 0, cur--, dest--, dest--
### 代码
```python
class Solution:
    def duplicateZeros(self, arr: List[int]) -> None:
        """
        Do not return anything, modify arr in-place instead.
        """
        n = len(arr)
        
        # 统计0的个数
        zeros = arr.count(0)
        
        # 从后往前遍历
        cur = n - 1          # 原数组指针
        dest = n + zeros - 1 # 虚拟数组指针
        
        while cur >= 0:
            if arr[cur] == 0:
                # 复写两个0
                if dest < n:
                    arr[dest] = 0
                dest -= 1
                if dest < n:
                    arr[dest] = 0
                dest -= 1
            else:
                # 复写非0元素
                if dest < n:
                    arr[dest] = arr[cur]
                dest -= 1
            cur -= 1
```
### 复杂度
时间复杂度: O(n)  
空间复杂度: O(1)
