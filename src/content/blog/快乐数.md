[202. 快乐数 - 力扣（LeetCode）](https://leetcode.cn/problems/happy-number/description/)
# 题意
快乐数的定义: 每一次将该数替换为它每位上的数字的平方和  
如此重复, 如果变为 1, 则是快乐数  
如果是无限循环变不到 1, 则不是快乐数
# 思路
最大的中间数是由 1999999999 转换而来的, `1+9^2*9=730`  
不会超过这个数  
因此用集合存的话, 也不会很大  
直接每次变化时, 将中间数存到集合里, 如果重复, 不是 1, 就不是快乐数  
重复, 是 1 就是快乐数

# 代码
```python
class Solution:
    def isHappy(self, n: int) -> bool:
        s=set()
        result=n
        while n!=1:
            s.add(n)
            digits=[int(d) for d in str(n)]
            n=sum(d**2 for d in digits)
            if n in s:
                return False
        
        return True
```
# 复杂度
时间复杂度: O(n)  
空间复杂度: O(1)
