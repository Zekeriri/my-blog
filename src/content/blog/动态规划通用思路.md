# 是什么
动态规划是一种有记忆的, 倒着推的解题方法
# 作用

问题满足以下条件, 可以用动态规划来解决
- 大问题的最优解必定包含子问题的最优解
- 遇到重叠子问题 (反复计算同一个子问题)
- 存在很多状态, 状态可以转换 (某阶段状态一旦确定, 后续决策只依赖当前状态, 不依赖之前怎么来的)
# 原理
是一种记忆化的递推, 可以说是递归的逆过程  
动态规划自底向上, 把中间结果存下来复用

# 模板
- 定义
- 状态转移方程
- 初始化
- 顺序
- 返回值
# 滚动数组优化
滚动数组就是当 dp[i] 只依赖固定数量的前几个状态时, 用覆盖替代保留, 把没用的数据滚走,  
可以实现空间复杂度降低一维
