[283. 移动零 - 力扣（LeetCode）](https://leetcode.cn/problems/move-zeroes/description/)

# 题意
将所有的 0 移到末尾, 非 0 的元素移到前面, 相对顺序不变
# 思路
## 1. 暴力
遍历, 当遍历到 0 时, 将之后的数往前移一位, 在末尾添加一个 0  
O(n^2)

## 2. 双指针
在暴力的基础上优化,  
用快速排序的思想, 数组划分区间的思想  
用一个 cur 来扫描整个数组, 另一个 dest 指针来记录非零数序列的最后一个位置,  
将 0:cur 的数组分为两部分, 0:dest 为非 0 序列, dest:cur 为 0 序列

### 步骤
1. 初始化 dest = -1 (在刚开始时还没有数, 为 -1), cur = 0
2. cur 开始遍历
	- 当 cur 指向的元素是非 0 时, dest++, 两个位置交换
	- 当 cur 指向的元素是 0 时, 直接略过

### 代码
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        dest = -1  # 指向非零序列的最后一个位置
        for cur in range(len(nums)):
            if nums[cur] != 0:
                dest += 1
                # 交换 nums[dest] 和 nums[cur]
                nums[dest], nums[cur] = nums[cur], nums[dest]
```

### 复杂度
时间复杂度: O(n)  
空间复杂度: O(1)
