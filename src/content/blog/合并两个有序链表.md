[21. 合并两个有序链表 - 力扣（LeetCode）](https://leetcode.cn/problems/merge-two-sorted-lists/description/)

# 1. 尾插法

遍历两个链表, 比较当前的两个节点, 将较小的节点插入新链表  
创建一个哨兵位, 方便插入操作

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        tail = dummy
        
        while list1 and list2:
            if list1.val <= list2.val:
                tail.next = list1
                list1 = list1.next
            else:
                tail.next = list2
                list2 = list2.next
            tail = tail.next
        
        # 直接连接剩余部分，不需要额外判断
        tail.next = list1 if list1 else list2
        
        return dummy.next
```

时间复杂度: O(n)  
空间复杂度: O(1)

# 2. 优先级队列

就是上面尾插法的优化, 先将所有链表的所有节点放入堆中  
这样就可以利用堆的特性, 将所有节点从小到大取出, 然后插入到新链表中

```python
class Solution:
    def mergeTwoLists(
        self, list1: Optional[ListNode], list2: Optional[ListNode]
    ) -> Optional[ListNode]:
        heap = []
        for node in [list1, list2]:
            while node:
                heapq.heappush(heap, node.val)
                node = node.next

        dummy = ListNode(0)
        tail = dummy
        while heap:
            tail.next = ListNode(heapq.heappop(heap))
            tail = tail.next
        return dummy.next
```

时间复杂度: O(n log n) 建堆的复杂度  
空间复杂度: O(n)

# 3. 递归法

自己尝试合并, 两个链表的头两个节点可以比大小, 可以知道最小的那一个节点为头节点, 那么这个头节点后面要接什么? 就是剩下的两个链表的合并  
这就出现了相同的问题, 规模会更小, 由于最后会到空, 一定会结束

把递归函数看成一个黑盒, 相信这个函数能解决问题

**递归函数设计**: `mergeTwoLists`
- 返回值: ListNode
- 参数: list1, list2
- 作用: 合并当前两个链表, 返回新的头节点

**递归函数流程**:
- 结束条件: 当一个为空时, 返回另一个节点
- 递推关系: 先比较当前两个节点的大小, 把小的作为头节点, 存储为返回值, 将该节点的 next 指向 mergeTwoLists 之后节点的结果
- 返回结果: 当前的头节点

```python
class Solution:
    def mergeTwoLists(
        self, list1: Optional[ListNode], list2: Optional[ListNode]
    ) -> Optional[ListNode]:
        if list1 is None:
            return list2
        if list2 is None:
            return list1
        if list1.val < list2.val:
            list1.next = self.mergeTwoLists(list1.next, list2)
            return list1
        else:
            list2.next = self.mergeTwoLists(list2.next, list1)
            return list2
```

时间复杂度: O(n)  
空间复杂度: O(1)
