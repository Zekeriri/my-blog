# 是什么
双指针就是定义两个变量（指针），在数组或链表上以某种规律移动，通过协同配合来解决问题
# 作用
双指针的作用就是“用两个变量协同扫描”，通过控制它们的移动速度和方向，把 O(n²) 的暴力遍历降维成 O(n) 的线性扫描。

# 原理
利用数据本身的“有序性”或“结构特性”，通过两个指针的相互约束，省去内层循环，将时间复杂度降一个维度。

# 应用场景
1. **对撞指针**：一左一右向中间靠拢，主要用于有序数组的查找，比如两数之和；
2. **快慢指针**：一快一慢同向走，主要用于链表，比如判断环、找中点；
3. **滑动窗口**：同向一前一后，中间夹着区间，主要用于子串或子数组问题。
