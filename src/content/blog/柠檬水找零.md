[860. 柠檬水找零 - 力扣（LeetCode）](https://leetcode.cn/problems/lemonade-change/description/)
# 题意
你一开始没有零钱, 需要从顾客那收到的钱来进行找零, 一杯柠檬水的价格为 5 美元, 顾客想你付的有 5, 10, 20 美元三种, 判断是否能正确找零
# 思路
贪心, 从小到大收钱,  
收到 5 不用找零  
收到 10 找零 5  
收到 20, 找零优先使用 10+5, 之后再用 5+5+5

## 原理
10 美元的作用有限, 只能用来找 20 美元  
而 20 美元不能用来找零  
所以按有用程度排 5 > 10 > 20  
按 5, 10, 20 的顺序来收  
而 20 美元找零时, 优先使用 10+5, 之后再用 5+5+5

# 代码
```python
class Solution:
    def lemonadeChange(self, bills: List[int]) -> bool:
        five = 0  # 手中5美元的数量
        ten = 0   # 手中10美元的数量
        
        for bill in bills:
            if bill == 5:
                five += 1
            elif bill == 10:
                if five == 0:
                    return False
                five -= 1
                ten += 1
            else:  # bill == 20
                # 优先使用10+5找零
                if ten > 0 and five > 0:
                    ten -= 1
                    five -= 1
                # 其次使用5+5+5找零
                elif five >= 3:
                    five -= 3
                else:
                    return False
        
        return True
```
# 复杂度
时间复杂度: O(n)  
空间复杂度: O(1)
