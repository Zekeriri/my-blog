[面试题 08.06. 汉诺塔问题 - 力扣（LeetCode）](https://leetcode.cn/problems/hanota-lcci/description/)

从 n=0,1,2 开始推演, 首先第一步就是要把最大的盘子给移到 C 处  
n=1 时, A=[0] 直接把 0 A->C  
n=2 时, A=[0,1] 把 0 A->B, 把 1 A->C, 此时 1 已经移到 C 处了, 再把 0 B->C  
n=3 时, A=[0,1,2], 把 0 A->B, 接下来只能 1 A->C, 此时 A=[2], B=[0], C=[1],  
继续 B->C, A=[2], B=[], C=[0, 1], 实际上 B 和 C 是等价的, 可以转化成 A=[2], B=[0, 1], C=[]  
继续 A->B 后, A=[], B=[0, 1], C=[2] 我们发现, 重新变成了把 [0, 1] 转移到另一个盘子, 变成了 n=2 的问题

这里就可以用递归来解决, 这个递归函数是一个黑盒, 我们不知道怎么实现, 但相信这个递归函数一定能解决问题, 所以先开始设计

**递归函数设计**: `move`
- 返回值: 空
- 参数: (n, source, target, temp)
- 作用: 把 source 的 n 个盘子移到 target, 中间盘子为 temp

然后开始思考流程, 把第 n 个盘子, 放到 target 处, 先要把 n-1 的盘子放到 temp 处  
就是先 move(n-1, source, temp, target),  
然后移动 n source->target  
此时 n 比其他要移的 n-1 个盘子大, 不会影响其中任意的盘子的移动, 把刚刚的 n-1 盘子 temp->target  
就是 move(n-1, temp, target, source) 就完成了

**递归函数流程**:
- 当 n <= 0 时, 就是移动 0 个以下的盘子, 什么也不用做, 直接返回
- move(n-1, source, temp, target)
- 移动 n source->target
- move(n-1, temp, target, source)

**代码**:
```python
class Solution:
    def hanota(self, A: List[int], B: List[int], C: List[int]) -> None:
        """
        Do not return anything, modify C in-place instead.
        """
        def move(n, source, target, temp):
            if n <= 0:
                return
            move(n-1, source, temp, target)
            target.append(source.pop())
            move(n-1, temp, target, source)
        move(len(A), A, C, B)
```

**复杂度**:  
- 时间 O(2^n): 一分为二, `T(n) = 2T(n-1) + 1`, `T(n) = 2^n - 1`
- 空间 O(n): 栈的最大深度为 n, 从 n-1->0
